Cambridge | The Open-Source Arcade Stacker

Cambridge is a block-stacking game (think Tetris) engine built for the modern age, as an alternative to popular engines such as NullpoMino. It is currently the most extensible and up-to-date single-player widely-known block-stacking game engine.
